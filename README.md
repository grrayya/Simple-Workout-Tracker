# Simple-Workout-Tracker

A basic app to track my workouts, ive made something similar before @-@ but react is rather efficent. 

### How to open the app

To run this on your computer and test it on your phone:

1. Download the code and run the setup command (`npm install`).
2. Start the app (`npx expo start`).
3. Scan the barcode that pops up using the Expo Go app on your phone.

### What it does right now

* Flips between "Upper Body" and "Lower Body" workout days.
* Records how much weight you lifted, how many times you lifted it, and how hard it felt.
* To keep things simple while I test the design, the buttons just add preset weights (like 135 lbs) instead of letting you type them in.
